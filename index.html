Setup e Avvio Immediato

Con Netiqo Framework puoi creare un server web self-hosted in pochissime righe di codice. L'host si configura automaticamente e gestisce routing, HTTPS e file statici.

Perfetto per applicazioni embedded, microservizi locali e server di sviluppo accessibili da tutta la rete Wi-Fi.

Program.cs
using Netiqo.Framework;

// Creazione dell'host
var host = new Host("127.0.0.1", 8080);

// Logging opzionale
host.LogMessage += Console.WriteLine;

// Registrazione route manuale
host.RegisterRoute(new Route("/", HttpMethod.GET, 
    htmlResponse: "<h1>Homepage</h1>"));

// Caricamento route da JSON
host.LoadRoutesFromJson("routes.json");

// Route tramite attributi
host.RegisterRoutesFromAttributes(new MyController());

// Avvio del server
await host.StartAsync();

Routing e Configurazione Avanzata

Scopri i diversi modi per configurare route e gestire HTTPS

routes.json & Controller
// routes.json
[
  {
    "Path": "/about",
    "Method": "GET",
    "HtmlResponse": "<h1>Chi siamo</h1>"
  }
]

// Controller con attributi
public class MyController 
{
    [Route("/login", HttpMethod.POST)]
    public string HandleLogin(string body)
    {
        var parsed = HttpUtility.ParseQueryString(body);
        return $"<h1>Welcome, {parsed["username"]}</h1>";
    }
}

// HTTPS con certificati
var host = new Host("127.0.0.1", 443, 
    "C:\\certs\\netiqo.pfx", "password");
