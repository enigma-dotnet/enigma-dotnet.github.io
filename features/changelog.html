v1.0.0

Initial Release
Released
  • New 🚀 Self-Hosted Lightweight - Create embedded or local web servers accessible via Wi-Fi or LAN without complex configurations
  • New 🛣️ Advanced Routing - Full support for GET/POST routing with logging from code, JSON, XML or attributes
  • New 🔐 HTTPS Integrated - Native support for SSL/TLS certificates with automatic configuration via SslStream
  • New 📁 Static Files - Automatic service of static files from the wwwroot/ folder for CSS, JS, images, and more
  • New ⚙️ Configurable Host - Flexible setup with customizable IP, port, certificates and logging for every need
  • New 🎯 Dynamic HTML - Static or dynamic HTML responses with custom handlers and form parsing
